# coffeeBuddy
This is a personal flutter project for learning purpose.

## Components Used

* Android Studio
* Firebase (FireFluter)
  - Firebase Authentication
  - Cloud Firestore
* Flutter
* Dart
* [Flutter Spinkit](https://pub.dev/packages/flutter_spinkit)
